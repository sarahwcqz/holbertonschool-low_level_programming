#Low-level programming

This repository contains the **Low-Level** project, part of Holberton School's **first trimester** curriculum. The project focuses on learning the C programming language, understanding memory management, pointers, data types, and basic algorithms. It also covers compiling, debugging, and writing efficient low-level code, providing a strong foundation for systems programming.

## Topics covered
- basics
- functions
- variadic functions
- loops
- nested loops
- arrays
- pointers
- linked-lists
- IO
- arguments
- malloc
- hash tables
- recursion
